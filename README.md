# 🐾 step mate
<img width="1548" height="870" alt="image" src="https://github.com/user-attachments/assets/dc2668fc-2a3e-47db-9e96-090704655b2b" />

걸음 수 기반 성장 시스템과 미니게임을 결합한 Wear OS 애플리케이션입니다.<br>
사용자의 활동량에 따라 캐릭터가 성장하고, 다양한 기능을 즐길 수 있습니다.<br>

## 1. 시작 화면
<img width="1962" height="1094" alt="image" src="https://github.com/user-attachments/assets/67c3b6ad-ba2b-450c-b4a4-fa91cd8d9e53" />

- 로그인 이후 최초로 진입하는 메인 화면입니다.<br>
- 캐릭터 상태 및 주요 기능으로 이동할 수 있습니다.<br>​

## 2. 상태창
<img width="1952" height="1090" alt="image" src="https://github.com/user-attachments/assets/39522c1d-db35-4231-8041-84ab5fc7bc49" />

- 캐릭터의 현재 상태를 확인할 수 있습니다.<br>
- 하루 1회 로그인 시 사과 아이템을 획득합니다.(효과: 경험치 +3) <br>​

## 3. 지구를 지켜라 (슈팅게임)
<img width="1952" height="1092" alt="image" src="https://github.com/user-attachments/assets/2f68aff6-643c-4ac7-8b07-f6789bc03f67" />

- 상태창에서 설정된 캐릭터 능력치를 기반으로 플레이합니다.<br>
- 플레이 횟수 제한 없이 자유롭게 이용 가능합니다.<br>
- 네트워크 연결 시 랭킹 시스템에 참여할 수 있습니다.<br>

## 4. 걸음 스트릭 및 내 정보​
<img width="1970" height="1096" alt="image" src="https://github.com/user-attachments/assets/43209a13-faf4-4561-8660-0e616fa93555" />

- 일별 걸음 수에 따라 활동량을 시각적으로 확인할 수 있습니다.<br>
- 걸음 수가 많을수록 색상이 더 진해집니다.<br>
- 각 날짜를 클릭하면 해당 일자의 상세 걸음 수를 확인할 수 있습니다.<br>

## 5. 모바일 App 지원
<img width="1572" height="884" alt="image" src="https://github.com/user-attachments/assets/670db9a8-a933-4f47-bd13-2ed7165b2155" />

- Watch에서 함께한 기록을 Mobile App에서도 확인할 수 있습니다.
- 블루투스로 연결된 Watch에 step mate app을 원격으로 설치할 수 있도록 지원합니다.
